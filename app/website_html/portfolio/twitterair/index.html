Twitter Air

Twitter Air is a desktop application based on Adobe AIR technology for twitter. User can use it to update twitter and view others. It's a platform independent application which means that you can run it on Windows PC, Mac and even GNU/Linux distribution.

I am very sorry that the badge installer will not work until the final version of AIR 2 is made publicly available. So please click here to install AIR 2 Beta first.
