jQuery treeselect

1. Summary

jQuery treeselect is a jquery plugin for create menu or navigation with ease.

Source Code on github: http://github.com/colorhook/jquery.treeselect

Download

2. Options

clickMode
If the clickMode is set to true which is default value, the trigger event of showing will be mouse click, or will be mouse over.
autoHide
If the autoHide is set to true which is default value, the trigger event of hidding will be mouse leave, or will be mouse click.
rootCls
The CSS class name of the root ul element.
rootZIndex
indicate the CSS z-index property of the root ul element.

$("#menu02").treeselect({
	clickMode:false,
	onChange:function(el){
		var elementA=el.find('a');
		$info.html(elementA.html()+" selected.");
	}
});
