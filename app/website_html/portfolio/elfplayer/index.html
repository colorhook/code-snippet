Elf Player

Elf Player is a Web Music Player based Adobe Flash technology, the following is feature list:

  • Sound spectrum display
  • Lyric synchronization
  • Lyric drap and drop

The source of Elf Player hosted under Google Code, for more information please visit the project home: http://code.google.com/p/elfplayer

Almost all of music played in this demo come from the 3rd sites, so the sound spectrum can't work because of the Security Sandbox.
